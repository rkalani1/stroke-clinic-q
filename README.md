# Stroke Clinic Pre-Visit Questionnaire

A structured pre-visit questionnaire that helps stroke and cerebrovascular clinic patients organize their history, symptoms, medications, and questions before an appointment.

**Live:** https://rkalani1.github.io/stroke-clinic-q/

> Educational tool only — not medical advice and not a substitute for care from a licensed clinician.

## Usage

`index.html` is a self-contained static page (no build step). Enable GitHub Pages (Settings → Pages) to serve it, or open the file directly in a browser.
